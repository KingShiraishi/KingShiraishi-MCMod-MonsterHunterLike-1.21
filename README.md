# KingShiraishi モンハン風Mod

Minecraft 1.21 の Mod です。

Mod作成の練習として、いま流行りの Monster Hunter Wilds を題材に選びました。  
なので `KingShiraishi-MCMod-MonsterHunterLike-1.21` という名前にしています。


# やったこと

- Mod専用のクリエイティブモードタブを追加
- 自作アイテム作成
- 自作ブロック作成
- 自作レシピ作成
- 自作モンスター作成
